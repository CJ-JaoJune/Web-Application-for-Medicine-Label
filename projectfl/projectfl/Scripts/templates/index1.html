    เทคโนโลยี OCR หรือ Optical Character Recognition
    
            
    
              OCR ย่อมาจาก "Optical Character Recognition" หรือ "การรู้จำอักษรด้วยทางทฤษฎีทางด้านทัศนศาสตร์" นั่นคือกระบวนการที่ใช้เทคโนโลยีคอมพิวเตอร์เพื่อแปลงข้อมูลที่ถูกพิมพ์หรือเขียนแล้วเป็นรูปแบบดิจิทัลที่สามารถอ่านและประมวลผลได้
            นอกจากนี้, OCR ยังสามารถแปลงข้อมูลจากภาพถ่ายหรือสแกนเอกสารเป็นข้อมูลที่สามารถใช้งานได้ในรูปแบบของข้อความ โดยโครงงานของเราพัฒนาขึ้นเพื่ออัปโหลดฉลากยา เนื่องจากในบางครั้งข้อมูลที่อยู่บนฉลากยามีขนาดเล็ก ซึ่งทำให้ยากต่อการอ่านข้อมูลในกรณีที่ผู้ใช้งานมีปัญหาทางสายตาหรือผู้สูงอายุ
            หรืออาจเกิดความสับสนในคำบนฉลากยาที่หลากหลายแต่มีความหมายเหมือนกัน และทำให้เกิดการเข้าใจผิด ผู้พัฒนาจึงพัฒนาขึ้นเพื่อแก้ปัญหาโดยการทดลองนําโมเดล You Only Look Once ver.8 ( YOLOv8 ) มาทํา กระบวนการ Optical Character Recognition (OCR) กับฉลากยา
            เพื่อช่วยเพิ่มความสะดวกในการใช้ยาของผู้ใช้งาน และช่วยในกระบวนการทำให้ผู้ใช้งานสามารถทำตามคำแนะนำการใช้ยาได้อย่างถูกต้อง

      You Only Look Once ver.8 ( YOLOv8 )
      
                
      
              
      • 
            
    
            
    
              ในปัจจุบันมีผู้ที่ประสบปัญหาบางครั้งข้อมูลที่อยู่บนฉลากยามีขนาดเล็ก ซึ่งทำให้ยากต่อการอ่านข้อมูลในกรณีที่ผู้ใช้งานมีปัญหาทางสายตาหรือผู้สูงอายุ และบางครั้งอาจเกิดความสับสนในคำบนฉลากยาที่หลากหลายแต่มีความหมายเหมือนกัน 
              ซึ่งปัญหาที่กล่าวมาข้างต้นอาจจะส่งผลทำให้ผู้ใช้งานไม่สามารถใช้งานยาได้อย่างถูกต้องตามคำแนะนำบนฉลากยาได้ 
              ด้วยปัญหาเหล่านี้จึงมีแนวทางการแก้ปัญหาโดยทดลองใช้โมเดล You Only Look Once ver.8 ( YOLOv8 ) โดยทั่วไปเป็นโมเดลสำหรับตรวจจับวัตถุ แต่ในโครงงานนี้ได้มีการนำโมเดลมาฝึกฝนด้วยชุดข้อมูลที่กำหนดเองเป็นการฝึกฝนให้ตรวจจับตัวอักษรให้ถูกต้อง 
              เพื่อศึกษาและวัดผลประสิทธิภาพของโมเดลในการทำกระบวนการ  Optical Character Recognition (OCR) เป็นกระบวนการแปลงไฟล์ภาพฉลากยาที่ได้รับการสแกนให้กลายเป็นไฟล์ข้อความเพื่อเพิ่มความสะดวกในการดึงข้อมูลที่สำคัญเกี่ยวกับยาใช้ยาของผู้ใช้งาน 
              และช่วยในกระบวนการทำให้ผู้ใช้งานสามารถทำตามคำแนะนำการใช้ยาได้อย่างถูกต้อง
